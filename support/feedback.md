# Edgefield Research Beta Feedback

Suggested beta feedback channels:

- Support email: support@edgefieldresearch.com
- Bug report form: add a Tally, Google Form, or GitHub issue link
- User feedback fields:
  - What were you trying to understand?
  - Which asset/read confused you?
  - Did the score audit explain the reading?
  - Was any source stale, missing, or unclear?
  - What would make the daily brief more useful?

Keep feedback focused on clarity, trust, and source integrity before adding new features.
