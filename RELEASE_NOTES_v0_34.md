# Release Notes — Macro Regime Scanner v0.34

**Version name:** Explainable Trust Layer  
**Base:** v0.33.1 Wide Release Calendar

## Summary

v0.34 is a conservative full iteration built from the working v0.33.1 baseline. It keeps the existing layout and live public-source stack, then adds the explainability features identified as the highest-impact way to raise the project rating.

## New

- Regime Queue Snapshot panel.
- Stronger score-audit panel in expanded asset rows.
- Driver pills for top positive and negative evidence.
- Context and excluded row examples.
- Asset-level “what this does not prove” caveat box.
- Sidebar trust guide.
- Markdown export: Current Regime Brief.

## Preserved

- v0.33.1 wide upcoming reports panel.
- Row toggle for non-applicable/display-only evidence.
- Source Health panel.
- Existing static deployment model.
- Existing orchestrator/scoring scripts.

## Tested

- `node --check app.js`
- `python scripts/recompute_live_scores.py`
- `python scripts/validate_data.py`
